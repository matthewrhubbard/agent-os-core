# Project Brief Template

Use this template to define the objective, context, constraints, stakeholders, risks, and success criteria for a project.

Related skill:
- `../20-skills/Strategy/Project-Intake.md`

## When To Use

Use this template when:

- Starting a new project
- Reframing a vague request into a defined project
- Aligning multiple people around the same scope and success criteria

## Quality Check

Before using the brief, confirm:

- The objective is specific
- Constraints are visible
- Success criteria are concrete
- Scope and non-goals are both stated
