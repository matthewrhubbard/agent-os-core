# Current Context Template

Use this template in the public Agent OS as the safe replacement for a live private priorities file.

This file should describe the current operating context without exposing sensitive personal details.

## Purpose

Capture what matters now so collaborators or AI tools can reason with the right context.

This is a strategic summary, not a full task tracker.

## Current Season

- What kind of season is this
- Why this season is different from a default operating mode

## Priority Stack

### Primary

- The single most important focus now

### Secondary

- `1-2` additional priorities only if they are real

## Non-Priorities

- What is intentionally not being optimized right now

## Key Success Markers

- What outcomes would show the current season is going well

## Constraints

- Time
- energy
- staffing
- money
- technical limitations
- external commitments

## Major Risks

- What could derail progress

## Active Open Decisions

- What still needs judgment

## Stop-Doing Rules

- What should be avoided during this season

## Review Cadence

- How often this file should be checked and refreshed

## Privacy Rule

If this file will be shared publicly:

- replace names with roles where possible
- remove medical, relationship, family, and financial specifics unless they are intentionally shareable
- keep only the context needed for sound decisions
