# Meeting Note Template

Use this template to save a meeting into a project as a durable Markdown artifact that AI tools, coding agents, and future project work can reference directly.

Related files:
- `../20-skills/Execution/Meeting-Prep.md`
- `../40-systems/Project-Notes-Convention.md`

## When To Use

Use this template when:

- A call, meeting, interview, or working session produced information worth keeping
- The notes should live with the project rather than inside a separate notes app
- You want one clean artifact that can be searched, linked, quoted, and reused

## Quality Check

Before saving the note, confirm:

- The title would still make sense six months from now
- The summary explains why the meeting matters, not just what was discussed
- Decisions and action items are clearly separated
- The file links back to the relevant project context
- Raw capture has been reduced to the minimum that is still useful
